# getunfccc

Script to download data from the UNFCCC. It utilizes [Selenium](
https://www.selenium.dev/), as the UNFCCC prevents simple downloads with a "security
firewall".

## background

Read more about the backstory here:

* <https://industrydecarbonization.com/news/why-is-it-needlessly-difficult-to-access-unfccc-emission-data.html>

If you just want to download the data itself, look here:

* <https://industrydecarbonization.com/docs/unfccc.html>
